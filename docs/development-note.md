## 2026-06-21 - Redis Pub/Sub 적용을 위한 학습 및 설계

### 적용 후보
- 파티 초대 이벤트
- 유저 따라가기 요청
- 파티원 입장/퇴장 알림
- 파티원의 착용 의상 변경 알림

### 설계 방향
Pub/Sub은 서버 간 실시간 이벤트 알림 용도로 사용  

> 단, Pub/Sub 메시지는 저장되지 않기 때문에    
> 유저 상태, 파티 상태, 의상 정보의 원본 데이터는 Redis Hash 또는 DB에 저장하고,   
> Pub/Sub은 변경 사실을 다른 로비 서버에 알려주는 용도로만 사용
